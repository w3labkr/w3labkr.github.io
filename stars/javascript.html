# Javascript

My favorite javascript repository

# Places

Turn any <input> into an address autocomplete

  • https://github.com/algolia/places (opens new window)

# Zoom

jQuery plugin for zooming images on mouseover.

  • https://github.com/jackmoore/zoom (opens new window)

# Zoomooz

An easy-to-use jQuery plugin for making zooming web pages.

  • https://github.com/jaukia/zoomooz (opens new window)

# Webfontloader

Web Font Loader gives you added control when using linked fonts via @font-face.

  • https://github.com/typekit/webfontloader (opens new window)

# Waypoints

Waypoints is a library that makes it easy to execute a function whenever you scroll to an element.

  • https://github.com/imakewebthings/waypoints (opens new window)

# Typed.Js

A JavaScript Typing Animation Library

  • https://github.com/mattboldt/typed.js/ (opens new window)

# Tilt.Js

A tiny 60+fps parallax tilt hover effect for jQuery.

  • https://github.com/gijsroge/tilt.js/ (opens new window)

# Type Scale

A Visual Type Scale Calculator

  • https://github.com/jeremychurch/type-scale (opens new window)

# Swiper

Most modern mobile touch slider with hardware accelerated transitions

  • https://github.com/nolimits4web/swiper (opens new window)

# Snowstorm

Scalable SNOMED CT Terminology Server using Elasticsearch

  • https://github.com/IHTSDO/snowstorm (opens new window)

# Slick

the last carousel you'll ever need

  • https://github.com/kenwheeler/slick (opens new window)

# Slick Lightbox

A lightbox wrapper for Ken's amazing slick carousel.

  • https://github.com/mreq/slick-lightbox (opens new window)

# SmoothState.Js

Unobtrusive page transitions with jQuery.

  • https://github.com/miguel-perez/smoothState.js (opens new window)

# Skrollr

Stand-alone parallax scrolling library for mobile (Android + iOS) and desktop. No jQuery. Just plain JavaScript (and some love).

  • https://github.com/Prinzhorn/skrollr (opens new window)

# Selectivizr

selectivizr is a JavaScript utility that emulates CSS3 pseudo-classes and attribute selectors in Internet Explorer 6-8.

  • https://github.com/keithclark/selectivizr (opens new window)

# Sheetjs

SheetJS Community Edition -- Spreadsheet Data Toolkit

  • https://github.com/SheetJS/sheetjs (opens new window)

# Select2

Select2 is a jQuery based replacement for select boxes. It supports searching, remote data sets, and infinite scrolling of results.

  • https://github.com/select2/select2 (opens new window)

# Respond

A fast & lightweight polyfill for min/max-width CSS3 Media Queries (for IE 6-8, and more)

  • https://github.com/scottjehl/Respond (opens new window)

# Jquery Pjax

pushState + ajax = pjax

  • https://github.com/defunkt/jquery-pjax (opens new window)

# JQuery Selectric

jQuery plugin for easy manipulation and customization of HTML selects

  • https://github.com/lcdsantos/jQuery-Selectric (opens new window)

# Jquery.Nicescroll

nicescroll plugin for jquery - scrollbars like iphone/ipad

  • https://github.com/inuyaksa/jquery.nicescroll (opens new window)

# Jquery.Tableparser

jQuery plugin to parse HTML table into Arrays

  • https://github.com/siuying/jquery.tableparser (opens new window)

# Lottie for Web

Render After Effects animations natively on Web, Android and iOS, and React Native.

  • https://github.com/airbnb/lottie-web (opens new window)

# Magnific Popup

Light and responsive lightbox script with focus on performance.

  • https://github.com/dimsemenov/Magnific-Popup (opens new window)

# Masonry

Cascading grid layout plugin

  • https://github.com/desandro/masonry (opens new window)

# Mathjs

An extensive math library for JavaScript and Node.js

  • https://github.com/josdejong/mathjs (opens new window)

# Medium Editor

Medium.com WYSIWYG editor clone. Uses contenteditable API to implement a rich text solution.

  • https://github.com/yabwe/medium-editor (opens new window)

# Moment

Parse, validate, manipulate, and display dates in javascript.

  • https://github.com/moment/moment (opens new window)

# Jquery Mousewheel

A jQuery plugin that adds cross-browser mouse wheel support.

  • https://github.com/jquery/jquery-mousewheel (opens new window)

# Isotope

Filter & sort magical layouts

  • https://github.com/metafizzy/isotope (opens new window)

# Imagesloaded

JavaScript is all like "You images done yet or what?"

  • https://github.com/desandro/imagesloaded (opens new window)

# Icheck

Highly customizable checkboxes and radio buttons (jQuery & Zepto)

  • https://github.com/dargullin/icheck (opens new window)

# Html5shiv

This script is the defacto way to enable use of HTML5 sectioning elements in legacy Internet Explorer.

  • https://github.com/aFarkas/html5shiv (opens new window)

# Highlight.Js

JavaScript syntax highlighter with language auto-detection and zero dependencies.

  • https://github.com/highlightjs/highlight.js/ (opens new window)

# GSAP

GreenSock's GSAP JavaScript animation library (including Draggable).

  • https://github.com/greensock/GSAP (opens new window)

# FullPage.Js

fullPage plugin by Alvaro Trigo. Create full screen pages fast and simple

  • https://github.com/alvarotrigo/fullPage.js/ (opens new window)

# Fullcalendar

Full-sized drag & drop event calendar

  • https://github.com/fullcalendar/fullcalendar (opens new window)

# Flickity

Touch, responsive, flickable carousels

  • https://github.com/metafizzy/flickity (opens new window)

# Clipboard.Js

Modern copy to clipboard. No Flash. Just 3kb gzipped

  • https://github.com/zenorocha/clipboard.js/ (opens new window)

# Barba

Create badass, fluid and smooth transition between your website's pages.

  • https://github.com/barbajs/barba (opens new window)

# AtvImg

This lightweight (2kb) plug-in will automatically turn your layered Apple TV PNGs into 3D parallax icons, the same way the new Apple TV treats app icons. You can have any number of AppleTV Images on your web page.

  • https://github.com/drewwilson/atvImg (opens new window)

# Anime

JavaScript animation engine

  • https://github.com/juliangarnier/anime (opens new window)

# Accounting.Js

A lightweight JavaScript library for number, money and currency formatting - fully localisable, zero dependencies.

  • https://github.com/openexchangerates/accounting.js/ (opens new window)

# Muuri

Infinite responsive, sortable, filterable and draggable layouts

  • https://gitlab.com/opensource5/javascript/muuri (opens new window)

# MorphingPageTransition

A simple morphing page transition effect where an SVG path gets morphed into another while the current page moves up.

  • https://github.com/codrops/MorphingPageTransition (opens new window)

# MorphingBackgroundShapes

A decorative website background effect where SVG shapes morph and transform on scroll.

  • https://github.com/codrops/MorphingBackgroundShapes (opens new window)

# Moment Timezone

Timezone support for moment.js

  • https://github.com/moment/moment-timezone (opens new window)

# Es6 Crawler Detect

This is an ES6 adaptation of the original PHP library CrawlerDetect, this library will help you detect bots/crawlers/spiders vie the useragent.

  • https://github.com/JefferyHus/es6-crawler-detect (opens new window)

# Cycle2

2nd gen cycling

  • https://github.com/malsup/cycle2 (opens new window)

# ScrollMagic

The javascript library for magical scroll interactions.

  • https://github.com/janpaepke/ScrollMagic (opens new window)

# PIE

A behavior for Internet Explorer allowing it to recognize and render various CSS3 box decoration properties

  • https://github.com/lojjic/PIE (opens new window)

# Picturefill

A responsive image polyfill for <picture>, srcset, sizes, and more

  • https://github.com/scottjehl/picturefill (opens new window)

# Particles.Js

A lightweight JavaScript library for creating particles

  • https://github.com/VincentGarreau/particles.js/ (opens new window)

# PapaParse

Fast and powerful CSV (delimited text) parser that gracefully handles large files and malformed input

  • https://github.com/mholt/PapaParse (opens new window)

# OwlCarousel2

DEPRECATED jQuery Responsive Carousel.

  • https://github.com/OwlCarousel2/OwlCarousel2 (opens new window)
